# Website Cheat Sheet
Documentation for ispconfig administrator about website management and view system log for fix other problem

# Table of Content
* ISPConfig
* Apache Log
* Website Log

## Apache Log
```bash
$ cd /var/log/apache2
$ tail access.log
$ tail error.log
```

## Website Log
```bash
$ cd /var/www/[Website]/log
$ tail error.log
```
